    Durian

    It's a fan-made tool for easier search of Path of Exile items. You can search by using PoE jargons like "6s5L chest 70life 60res".

    It also features an automated search that will notify you (by playing a sound) if any items are found.

    See more details in:
    • Reddit thread
    • PoE forum thread

    There's also the the help page for a reference of all the search terms available.
